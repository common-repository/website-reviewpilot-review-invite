Deze plugin van Review pilot geeft u de mogelijkheid direct uw klanten een uitnodiding voor beoordelingen te sturen. Ook geeft deze plugin de mogelijkheid de beoordelingen direct te plaatsen op uw webshop.


Deze plugin wordt gebruikt om je Wordpress installatie te verbinden met het Review Pilot platform.
M.b.v. deze plugin is het mogelijk je reviews op te halen en weer te geven op elke gewenste pagina m.b.v. een shortcode.
Tevens is het mogelijk direct je gepersonaliseerde Review Pilot uitnodigingsmail te verzenden naar een e-mailadres direct vanuit je Wordpress installatie.

De Rievew Pilot Wordpress plugin maakt gebruik van 4 externe links om de diensten uit te kunnen voeren:

- https://reviewpilot.nl/key_verification/magento
- https://reviewpilot.nl/key_verification/send_review_invite
- https://reviewpilot.nl/key_verification/insert_invited_customer
- https://reviewpilot.nl/key_verification/getAllReviews

Deze links worden gebruikt voor het "ophalen van reviews vanuit reviewpilot.nl", "Het versturen van een uitnodigingsmail naar e-mailadressen", "koppeling met Magento gebruikers plugin" en "je uitnodiding op te slaan in je reviewpilot.nl account".
De bovenstaande informatie kan enkel worden opgehaald met een reviewpilot.nl API key van de gekoppelde website.
